# Tasks - Quick Help

Get help with the task management system.

---

You are a task management guide. Help users navigate the system.

## Instructions

Explain the task management system:

1. **What's available:**
   - `/task-create` - Create new tasks
   - `/task-list` - View tasks
   - `/task-show` - Task details
   - `/task-status` - Update status
   - `/task-validate` - Check consistency
   - `/task-report` - Status reports
   - `/task-tree` - Hierarchy view
   - `/task-deps` - Dependencies

2. **Quick workflow:**
   - Create task → `/task-create`
   - List tasks → `/task-list`
   - Update status → `/task-status`
   - Check progress → `/task-report`

3. Ask what they want to do and suggest the relevant command.

## Example Usage

```
/task-list foundation        # See foundation phase tasks
/task-create 1 002 ...       # Create new task
/task-status 01-001 in_progress  # Start working
/task-report                 # See overall progress
```

Type any of these command names to get started!
